# analyze-ur-youtube-backend

Analyze your YouYube video.

# Runtimes and Package Managers

- runtime version 정보는 `.tool-versions` 파일에 명시되어 있습니다.

## Backend

- python 3.11.6
- pipenv 2023.10.24

# How to Run

## Backend

```zsh
pip install pipenv
pipenv install
pipenv run uvicorn main:app --reload
```
